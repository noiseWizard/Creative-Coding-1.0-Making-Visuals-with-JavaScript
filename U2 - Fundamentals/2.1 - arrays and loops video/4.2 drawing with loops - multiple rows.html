rows of squares

multiple rows can be created using nested loops:

        
    for (i = 0; i < 5; i++) {

        for (j = 0; j < 5; j++) {
            let width = 60;
            let height = 60;
            let gap = 20;
            let x = 100 + (width + gap) * i;
            let y = 100 + (width + gap) * j;

            rowContext.beginPath();
            rowContext.rect(x, y, width, height)
            rowContext.stroke();
        }

    }

traditionally, if a second index is needed in a loop, the letter j is used

squares in squares

and squares can be drawn in squares:

        
    for (i = 0; i < 5; i++) {

        for (j = 0; j < 5; j++) {
            let width = 60;
            let height = 60;
            let gap = 20;
            let x = 100 + (width + gap) * i;
            let y = 100 + (width + gap) * j;

            squaresInContext.beginPath();
            squaresInContext.rect(x, y, width, height)
            squaresInContext.stroke();

            squaresInContext.beginPath();
            squaresInContext.rect(x + 8, y + 8, width - 16, height - 16);
            squaresInContext.stroke();
        }

    }
