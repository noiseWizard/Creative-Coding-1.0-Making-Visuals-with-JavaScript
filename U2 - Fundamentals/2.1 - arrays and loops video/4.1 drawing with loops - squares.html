this loop draws five squares in a row because the loop increases the horizontal starting position with each run of the loop

the loop code:

        
    for (let i = 0; i < 5; i++) {
        parallelContext.beginPath();
        parallelContext.rect(100 + 60 * i, 100, 60, 60);
        parallelContext.stroke();
    }
        
    
        
        the 100 + 60 * i means:
        horizontal draw location + (move amount * current value of i)
        for each of the five, the location value would be
            100 = 100 + (60 * 1) (it's not starting at 160 because the box is drawn from top left and the box size is 60)
            100 = 100 + (60 * 2)
            100 = 100 + (60 * 3)
            100 = 100 + (60 * 4)
            100 = 100 + (60 * 5)
